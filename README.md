# About the Dataset

This dataset provides a detailed overview of gym members' exercise routines, physical attributes, and fitness metrics.

It contains **973 gym member records**, including key performance indicators such as heart rate, calories burned, and workout duration. Each entry also includes demographic information and experience levels, enabling comprehensive analyses of fitness patterns, athlete progression, and health trends.

## Dataset Features

| Feature | Description |
|---------|-------------|
| **Age** | Age of the gym member. |
| **Gender** | Gender of the gym member (*Male* or *Female*). |
| **Weight (kg)** | Member's weight in kilograms. |
| **Height (m)** | Member's height in meters. |
| **Max_BPM** | Maximum heart rate (beats per minute) during workout sessions. |
| **Avg_BPM** | Average heart rate during workout sessions. |
| **Resting_BPM** | Heart rate at rest before the workout. |
| **Session_Duration (hours)** | Duration of each workout session in hours. |
| **Calories_Burned** | Total calories burned during each session. |
| **Workout_Type** | Type of workout performed (e.g., Cardio, Strength, Yoga, HIIT). |
| **Fat_Percentage** | Body fat percentage of the member. |
| **Water_Intake (liters)** | Daily water intake during workouts. |
| **Workout_Frequency (days/week)** | Number of workout sessions per week. |
| **Experience_Level** | Experience level, ranging from beginner (**1**) to expert (**3**). |
| **BMI** | Body Mass Index, calculated from height and weight. |

## Applications

This dataset is ideal for:

- 📊 Data science and machine learning projects.
- 🏥 Health and fitness research.
- 💪 Exercise habit analysis.
- 📈 Modeling fitness progression.
- ❤️ Studying relationships between demographic and physiological variables.
- 🔥 Analyzing factors that influence workout intensity, endurance, calorie expenditure, and overall health.

Overall, the dataset offers a comprehensive foundation for exploring how different physical, physiological, and behavioral factors contribute to fitness performance and well-being.
